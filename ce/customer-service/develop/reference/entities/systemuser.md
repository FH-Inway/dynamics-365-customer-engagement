# User (SystemUser) table/entity reference

Person with access to the Microsoft CRM system and who owns objects in the Microsoft CRM database.

> [!NOTE]
> The Microsoft Dynamics 365 Customer Service User (SystemUser) table extends the [Microsoft Dynamics 365 User (SystemUser) table](/dynamics365/developer/entities/systemuser).


## Messages

The following table lists the messages for the User (SystemUser) table.
Messages represent operations that can be performed on the table. They may also be events.

| Name <br />Is Event? |Web API Operation |SDK for .NET |
| ---- | ----- |----- |
| `msdyn_GDPROptoutUser`<br />Event: True |**msdyn_GDPROptoutUser action** |[Learn to use messages with the SDK for .NET](/power-apps/developer/data-platform/org-service/use-messages)|
