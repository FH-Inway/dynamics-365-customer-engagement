# Key Vault Reference (KeyVaultReference) table/entity reference

Contains data that refers to an Azure Key Vault containing credentials used to connect to secure web-hosted resources.

> [!NOTE]
> The Microsoft Dynamics 365 Key Vault Reference (KeyVaultReference) table extends the [Microsoft Dataverse Key Vault Reference (KeyVaultReference) table](/power-apps/developer/data-platform/reference/entities/keyvaultreference).
