# Data Movement Service Request (msdyn_dmsrequest) table/entity reference

Table for Data Movement Service Requests

> [!NOTE]
> The Microsoft Dynamics 365 Data Movement Service Request (msdyn_dmsrequest) table extends the [Microsoft Dataverse Data Movement Service Request (msdyn_dmsrequest) table](/power-apps/developer/data-platform/reference/entities/msdyn_dmsrequest).

The following table lists the events for the Data Movement Service Request (msdyn_dmsrequest) table.
Events are messages that exist so that you can subscribe to them. Unless you added the event, you shouldn't invoke the message, only subscribe to it.
