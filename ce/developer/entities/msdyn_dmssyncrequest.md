# DMS Sync Request (msdyn_dmssyncrequest) table/entity reference

An entity to save DMS sync requests.

> [!NOTE]
> The Microsoft Dynamics 365 DMS Sync Request (msdyn_dmssyncrequest) table extends the [Microsoft Dataverse DMS Sync Request (msdyn_dmssyncrequest) table](/power-apps/developer/data-platform/reference/entities/msdyn_dmssyncrequest).


## Messages

The following table lists the messages for the DMS Sync Request (msdyn_dmssyncrequest) table.
Messages represent operations that can be performed on the table. They may also be events.

| Name <br />Is Event? |Web API Operation |SDK for .NET |
| ---- | ----- |----- |
| `Restore`<br />Event: True |<xref:Microsoft.Dynamics.CRM.Restore?displayProperty=nameWithType /> |[Learn to use messages with the SDK for .NET](/power-apps/developer/data-platform/org-service/use-messages)|
