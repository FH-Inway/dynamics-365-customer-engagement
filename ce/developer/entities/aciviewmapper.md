# ACIViewMapper table/entity reference

Customized messages for an entity that has been renamed.

> [!NOTE]
> The Microsoft Dynamics 365 ACIViewMapper table extends the [Microsoft Dataverse ACIViewMapper table](/power-apps/developer/data-platform/reference/entities/aciviewmapper).


## Messages

The following table lists the messages for the ACIViewMapper table.
Messages represent operations that can be performed on the table. They may also be events.

| `Restore`<br />Event: True |<xref:Microsoft.Dynamics.CRM.Restore?displayProperty=nameWithType /> |[Learn to use messages with the SDK for .NET](/power-apps/developer/data-platform/org-service/use-messages)|
