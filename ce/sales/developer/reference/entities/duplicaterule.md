# Duplicate Detection Rule (DuplicateRule) table/entity reference

Rule used to identify potential duplicates.

> [!NOTE]
> The Microsoft Dynamics 365 Sales Duplicate Detection Rule (DuplicateRule) table extends [Duplicate Detection Rule (DuplicateRule) table (Microsoft Dataverse)](/power-apps/developer/data-platform/reference/entities/duplicaterule) and [Duplicate Detection Rule (DuplicateRule) table (Microsoft Dynamics 365)](/dynamics365/developer/reference/dataverse/entities/duplicaterule).
>
> This reference contains only the changes added by Microsoft Dynamics 365 Sales.
> See the [Duplicate Detection Rule (DuplicateRule) table (Microsoft Dataverse)](/power-apps/developer/data-platform/reference/entities/duplicaterule) and [Duplicate Detection Rule (DuplicateRule) table (Microsoft Dynamics 365)](/dynamics365/developer/reference/dataverse/entities/duplicaterule) reference documentation for more information.
